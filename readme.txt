== Description ==

*Simple Bookings* is a plugin that allows you to manage booking request approvals. 

**Plugin Features:**

* Easily add Booking Requests using the Booking menu.
* View booking request approval status.
* E-mail triggers go to the user completing the booking request, and to the user who is assigned to approving the request.

== Installation ==

1. Upload the 'Simple Bookings' plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit Dashboard / Booking menu to Add Requests or View Requests.

== Frequently Asked Questions ==

= How do I add a booking? =

Login to the Dashboard and go to the Bookings / New Request area or add the [booking] shortcode to any page or post to display a front-end booking form.  Enter a title and description of your booking, and the start and end times and dates.  Choose who should approve the booking from the pull-down menu.

== Notes ==
1. The Bookings menu is viewable by all roles.  That said, only Admins and Editors and see and edit and approve requests.  Subscribers can only see their own Booking Requests.
